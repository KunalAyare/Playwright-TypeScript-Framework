# Playwright TypeScript Framework

## About This Project
Automation framework built using Playwright and TypeScript with Page Object Model (POM) structure.

## Application Under Test
Saucedemo - https://www.saucedemo.com

## Framework Structure
- `pages/` - Page Object classes for each page
- `tests/` - Test files for each feature

## Pages Covered
- Login Page
- Inventory Page
- Cart Page

## Tests Covered
- Login with valid credentials
- Add item to cart
- Verify cart and checkout button

## Tech Stack
- Playwright
- TypeScript
- Page Object Model (POM)

## How to Run Tests
```
npx playwright test
```
